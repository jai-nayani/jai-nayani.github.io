# Apercu Font Files

## How to Add Apercu Font

1. **Purchase Apercu** from [MyFonts](https://www.myfonts.com/collections/apercu-font-colophon-foundry) or Colophon Foundry
   - You'll need a **Webfont license** for use on your website
   - The font comes in multiple weights: Light, Regular, Medium, Bold (each with italics)

2. **Download the webfont files** (.woff2 and .woff formats)

3. **Place the font files** in this folder (`/public/fonts/apercu/`) with these exact names:
   - `Apercu-Light.woff2` and `Apercu-Light.woff`
   - `Apercu-LightItalic.woff2` and `Apercu-LightItalic.woff`
   - `Apercu-Regular.woff2` and `Apercu-Regular.woff`
   - `Apercu-Italic.woff2` and `Apercu-Italic.woff`
   - `Apercu-Medium.woff2` and `Apercu-Medium.woff`
   - `Apercu-MediumItalic.woff2` and `Apercu-MediumItalic.woff`
   - `Apercu-Bold.woff2` and `Apercu-Bold.woff`
   - `Apercu-BoldItalic.woff2` and `Apercu-BoldItalic.woff`

4. **The font will automatically be applied** to your portfolio once the files are in place!

## Note

- Only add font files you have a license for
- The CSS is already configured to use Apercu - just add the font files
- If font files are missing, the site will fall back to system fonts
